# Keyboard Shortcuts Reference

Complete guide to all keyboard shortcuts in Replay Studio. Press `?` in the application to see this information in a quick-reference modal.

## Drawing Tools

Select different drawing tools without clicking the toolbar.

| Shortcut | Tool | Description |
|----------|------|-------------|
| `V` | Select | Select, move, and modify existing annotations |
| `P` | Pen | Freehand drawing tool for sketching |
| `L` | Line | Draw straight lines between two points |
| `A` | Arrow | Draw arrows with automatic arrowheads |
| `R` | Rectangle | Draw rectangular shapes |
| `C` | Circle | Draw circles and ellipses |
| `T` | Text | Add text annotations to the video |
| `S` | Spotlight | Highlight specific areas with a spotlight effect |
| `M` | Magnifier | Create magnification zones for detailed analysis |
| `K` | Tracker | Add player/object tracking markers with crosshairs |

**Pro Tip**: These shortcuts work from anywhere in the application, even while playing video.

## Video Playback

Control video playback precisely without touching your mouse.

### Basic Playback

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Space` | Play/Pause | Toggle between playing and paused states |
| `Home` | Go to Start | Jump to the beginning of the video (0:00) |
| `End` | Go to End | Jump to the end of the video |

### Frame-by-Frame Navigation

| Shortcut | Action | Description |
|----------|--------|-------------|
| `←` (Left Arrow) | Previous Frame | Step backward one frame (~33ms at 30fps) |
| `→` (Right Arrow) | Next Frame | Step forward one frame (~33ms at 30fps) |

**Note**: Frame duration is calculated as 1/30th of a second (assuming 30fps video).

### J-K-L Professional Editing

Industry-standard video editing shortcuts inspired by professional NLE software.

| Shortcut | Action | Description |
|----------|--------|-------------|
| `J` | Step Backward | Jump backward ~100ms (3 frames) |
| `K` | Pause | Pause video playback immediately |
| `L` | Step Forward | Jump forward ~100ms (3 frames) when paused |

**Usage Pattern**:
- Press `K` to pause at any moment
- Tap `J` repeatedly to review backward
- Tap `L` repeatedly to advance forward
- This gives you precise control without using the timeline

## In/Out Points

Mark specific sections of video for analysis or export.

| Shortcut | Action | Description |
|----------|--------|-------------|
| `I` | Set In Point | Mark the current time as the In point (start) |
| `O` | Set Out Point | Mark the current time as the Out point (end) |
| `[` | Jump to In | Jump playhead to the In point |
| `]` | Jump to Out | Jump playhead to the Out point |

**Workflow Example**:
1. Play to desired start → Press `I`
2. Play to desired end → Press `O`
3. Jump between with `[` and `]` to verify
4. Export only this range if desired

## Editing and Annotation Management

Modify and manage your annotations efficiently.

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+Z` | Undo | Undo the last annotation or change |
| `Ctrl+Y` | Redo | Redo the last undone action |
| `Ctrl+A` | Select All | Select all annotations on the canvas |
| `Delete` | Delete Selected | Remove currently selected annotation(s) |
| `Escape` | Deselect / Select Tool | Clear selection or switch to Select tool |

**Note**: On macOS, use `Cmd` instead of `Ctrl`.

## Advanced Techniques

### Rapid Tool Switching

Switch between tools quickly during annotation:

1. Press `P` to grab pen and draw
2. Press `V` to switch to select and adjust
3. Press `A` to add an arrow
4. Press `T` to label with text

All without touching your mouse!

### Precise Frame Navigation

For ultra-precise work:

1. Press `K` to pause
2. Use `←` and `→` to step frame-by-frame
3. Press `P` to draw at exact moment
4. Press `Space` to resume and review

### Scrubbing Workflow

Professional video review technique:

1. Press `K` to pause
2. Tap `J` multiple times to review backward
3. When you find the moment, tap `L` to advance slowly
4. Mark with `I` when you find the exact frame
5. Continue with `L` to find end point
6. Mark with `O`

### Annotation Layer Management

Control which annotations are in front:

1. Latest annotations appear on top automatically
2. Use Select tool (`V`) to click and bring forward
3. Use `Delete` to remove unwanted annotations
4. Use `Ctrl+Z` to undo and reorder

## Platform Differences

### Windows
- Use `Ctrl` for modifier key
- Use `Alt` for alternative actions
- Standard Windows keyboard behavior

### macOS
- Use `Cmd` instead of `Ctrl` for most shortcuts
- Use `Option` instead of `Alt`
- Standard macOS keyboard behavior

### Linux
- Use `Ctrl` for modifier key
- Follows standard Linux desktop conventions

## Tips for Memorization

### Start with Essentials

Learn these first:
1. `Space` - Play/Pause
2. `P` - Pen tool
3. `V` - Select tool
4. `Ctrl+Z` - Undo
5. `?` - Show shortcuts help

### Tool Mnemonics

Remember tools by their first letter:
- **P**en, **L**ine, **A**rrow, **R**ectangle, **C**ircle, **T**ext
- **S**potlight, **M**agnifier
- **V** for "selection arrow" (like pointer cursor)
- **K** for "tracKer"

### Create Muscle Memory

Practice common sequences:
1. `I` → play → `O` (set range)
2. `P` → draw → `V` → adjust (draw and tweak)
3. `K` → `L` → `L` → `I` (find exact moment)
